# 📚 Python AI Learning: Local PDF Assistant

这是一个基于 **Python** 和 **DeepSeek 大模型** 实现的本地 PDF 智能助手。它可以自动读取本地 PDF 文档内容，并通过 AI 对文档进行总结、分析和问答。

🚀 **在线体验地址**: [https://python-ai-learning-7b6gkwrmbaprq9wfnzifsm.streamlit.app/]

---

## 🌟 核心功能

- **PDF 文字解析**：基于 `pypdf` 实现对原生 PDF 文本的精准提取。
- **智能问答**：集成 **DeepSeek-V3** 接口（兼容 OpenAI 标准），实现深度语义理解。
- **配置分离**：使用 `.env` 环境变量管理 API Key，确保代码安全性。
- **版本控制**：标准的 Git 流程管理，内置 `.gitignore` 防止隐私泄露。

---

## 🛠️ 技术栈

- **语言**: Python 3.10+
- **环境管理**: Conda (Miniconda3)
- **核心库**: 
  - `openai`: 用于调用大模型接口
  - `pypdf`: 用于处理 PDF 文本
  - `python-dotenv`: 用于加载环境变量
- **大模型**: DeepSeek-V3
